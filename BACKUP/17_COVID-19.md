## Web3

今天看了Dapp大概的开发流程，传统应用程序开发通常由以下几个部分组成

1. API/应用程序服务器（REST/GraphQL）
2. 认证层
3. 数据库
4. 客户端框架、平台和库
5. 文件存储

而Dapp也类似

- 去中心化的网络基础设施
- 拥有（数据、内容和平台）的所有权
- 原生数字支付
- 自主的身份
- 分布式的、无信任的、强大的基础设施
- 开放、公共、可组合的后端
- 预言机

## 金融

看了现代经济的运作（还是很难理解）

+ 一个人的支出是另一个人的收入
+ 外汇(65%) + 商业银行债权(35%) + 对政府债权(5%) + 其他资产(x%) - 政府存款(10%) = 基础货币的信用保证

这是人民币的信用体系架构，这样来看的话，人民币走强简直讲笑话呢。

如果要结合世界其他货币的话，就更复杂了，总之人民币很封闭，不会被砸盘。

毕竟中国曾经可是在与世隔绝的情况下活了400多年呢。

### 2022年可能的发展方向

从2012年开始已经倒退发展10年了，今天看了下22年可能预测的发展方向

1. 混合工作安排将继续存在

   这个在海外非常明显啊，国内的外企也是，remote的岗位越来越多，人们开始习惯remote的模式，且效率同样很高，我认为想让一个习惯了remote的人重回以前的默认很困难。

2. 人工智能出现在比以往更多的地方

   目前的什么大数据，推荐算法（最tm恶心的发明）一般都是通过nlp或者ai来实现，22年卫生部门的人工智能可能会迎来较大的发展

3. 软件的自动化提高生产力

   这个几乎都在讲机器人，完全不懂了

   + 机器人过程自动化（RPA）代替重复性劳动  -》贵
   + 机器人即服务（RAAS）基于订阅等方式，适用于小公司
   + 自主移动机器人（AMR）自动执行任务

4. 沉浸式游戏 VR/AR metaverse

   不太看好

5. 5G无缝体验，边缘计算

   不懂

6. 高速互联网即将覆盖农村

   国内基建还不够匀质，等starlink了，如果用得起的话

7. 有助于拯救地球的可持续能源解决方案

   + 人工智能和大数据
   + 先进的机器人

   能源确实是越来越重要的东西
